# 계약서·제안서(IM) 비교 도구

증권사·은행이 보낸 **계약서**와 **제안서(IM)**를 업로드해서
금융조건과 채권보전조치의 공통점/차이점을 찾고,
계약서를 기준으로 제안서를 어떻게 수정해야 하는지 정리해 주는 도구입니다.

> ⚠️ 결과는 법적 판단이 아니라 **검토 보조용**입니다.

## 현재 진행 상황
- [x] 1단계: 계약서 텍스트 추출 (페이지 번호 포함)
- [ ] 2단계: 제안서(IM) 텍스트 추출
- [ ] 3단계: 핵심 내용 찾기 + 분할 화면
- [ ] 4단계: 비교 및 정리

## 실행 방법
```bash
# 1) 패키지 설치 (처음 한 번만)
pip install -r requirements.txt

# 2) 실행
streamlit run app.py
```

## 기술 스택
- Python / Streamlit
- pdfplumber (텍스트+위치 추출)
- PyMuPDF (페이지 이미지 캡처) — 3단계에서 사용
- docx2pdf / LibreOffice (워드 → PDF 변환)
- OpenAI API (내용 분석) — 3·4단계에서 사용
